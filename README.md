# Physical-Therapy-Clinic-Operations-Performance-Dashboard
End-to-end Power BI business intelligence project analyzing patient retention, acquisition, provider utilization, referral performance, appointment volume, insurance mix, and revenue KPIs for a physical therapy clinic. Includes data modeling, DAX calculations, interactive dashboards, and executive-level reporting.


## Project Overview

This project is a multi-page Power BI dashboard built for a fictional outpatient physical therapy clinic. The goal was to transform operational and financial data into actionable insights that support decision-making across patient management, provider performance, scheduling, referrals, and revenue.

The dashboard was designed to simulate a real-world reporting environment where clinic managers and leadership teams need visibility into key performance indicators (KPIs) without manually reviewing spreadsheets or exporting reports.

---

## Business Questions

This dashboard was built to answer questions such as:

* How many patients are new versus returning?
* Are patient retention and acquisition trends healthy?
* Which referral sources generate the most patient volume?
* Which providers are handling the most appointments and revenue?
* How does appointment volume vary throughout the week?
* Which insurance payers contribute the most revenue?
* How effectively is the clinic converting visits into revenue?
